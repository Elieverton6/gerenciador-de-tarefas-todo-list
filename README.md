# gerenciador-de-tarefas-todo-list
Este projeto é um Sistema de Gerenciamento de Tarefas desenvolvido com o framework Laravel (PHP). Ele permite a criação, finalização, reversão da finalização e exclusão de tarefas de maneira intuitiva e eficiente. O sistema também integra o uso de Cache para otimizar o desempenho, tornando as leituras de tarefas mais rápidas.
